Git: Основы версионирования

Git config —global - указываем электронную почту и имя. Вызвать можно из любой директории

Репозитория - местов котором хранятся все изменении проекта

Git init - делает папку с файлом git-репозиторием

Rm -rf .git - различить папку если что-то пошло не так

Ключ R - позволяет удалять папки вместе с их содержимым, рекурсивно

Ключ F - избавит от вопрос действительно это нужно удалить или нет

Git status - текущее состояние репозитория

Git add - сохранения файла в новой версии

Git add —all - добавит несколько файлов и подготовит их к сохранению разом
